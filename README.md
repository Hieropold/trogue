# Trophyroom

Command line tool to view the PSN trophies.

# Configure

Configuration is done through environment:
* `TROPHYROOM_STEAM_API_KEY`
* `TROPHYROOM_STEAM_ID`

# Build

To build the tool:
```
cargo build
cargo build --release
```

or

```
cargo run
```

To format code:
```
cargo fmt
```

To run linting:
```
cargo clippy
```
